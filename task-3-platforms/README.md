# Task 3 — Coding Platform Exploration

## Objective

The objective of this task was to explore coding platforms and improve problem-solving skills using online tools.

---

## Platform Used

HackerRank

## Description

In this task, an account was created on HackerRank, and beginner-level coding problems were solved to improve programming skills.

Additionally, a Google Form was created to promote digital literacy awareness among students. The form included questions related to safe internet usage, cybersecurity awareness, and responsible online behavior.

Screenshots of the HackerRank profile and Google Form were captured.

---

## Outcome

This task helped improve logical thinking, coding skills, and awareness about using digital tools for collecting responses and spreading information.
